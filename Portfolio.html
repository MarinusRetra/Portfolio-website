  • Text Rpg

    Als speler ben je een gevangene, om te ontsnappen moet je door een set willekeurige gegenereerde scenario's heen die kunnen rijken van vijanden bevechten tot het oplossen van puzzels.
    Features:
    Een character creator , twee soorten puzzels, turn-based combat, combat skills, xp en leveling, vijand en speler komen uit één klasse, moeilijkheids selectie en een tekst snelheid optie.
    Verdere features en hoe je moet spelen staan in het readme document op Github.

  • GunBot (Demo)

    GunBot is een 2D Singleplayer Run and gun game waarin de speler meerdere unieke wapens gebruikt om door de vijanden heen te vechten tot de speler het einde van elk level bereikt heeft. GunBot is nu alleen nog een demo maar ik ben van plan Gunbot een volledige game te maken.

    Features die nu in de game te vinden zijn:
    Meerdere wapens die de speler kan gebruiken.
    3 Verschillenden vijanden.
    Alle vijanden hebben unieke bewegings- en schietpatronen.
    Een jumpbuffer, walljump en coyote time.
    Alle wapens hebben hun eigen schietpatroon.

    Komende features:
    Meer levels.
    Meer wapens voor de speler.
    Meer soorten vijanden.
    Vijanden die naar de speler toe bewegen.
    Bosses.
    Meer geluidseffecten.
    Movement abilities die door de game heen te vinden zijn.

  • Back To Your Roots

    in ons eerste jaar aan Noorderpoort hebben Mike Oomen en ik meegedaan aan de 2023 global gamejam groningen. Gedurende de 3 dagen van de gamejam hebben we Back To Your Roots gemaakt dit is een korte 2D game waard je doel is om zo ver mogelijk naarbeneden te komen terwijl je onderweg een paar puzzels moet oplossen. De pixel art en animaties zijn door mij gemaakt, ik heb ook een paar dingen aan programmeer werk gedaan maar mijn focus lag tijdens deze gamejam op het maken van pixel art.
