Projects

  • Text Rpg

    Als speler ben je een gevangene, om te ontsnappen moet je door een set willekeurige gegenereerde scenario's heen die kunnen rijken van vijanden bevechten tot het oplossen van puzzels.
    Features:
    Een character creator, twee soorten puzzels, turn-based combat, combat skills, xp en leveling, vijand en speler komen uit één klasse, moeilijkheids selectie en een tekst snelheid optie.
    Verdere features en hoe je moet spelen staan in het readme document op Github.
